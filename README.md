# Simple Calculator 🧮

A clean, responsive, and functional web-based calculator built from scratch to practice front-end development and logic implementation.

## 🚀 Features
- **Clean UI:** Minimalist design for a great user experience.
- **Responsive:** Works seamlessly on both desktop and mobile browsers.
- **Core Math:** Supports all standard arithmetic operations.
- **Error Handling:** Gracefully handles invalid inputs like division by zero.

## 🛠 Tech Stack
- **HTML5:** Structure and content.
- **CSS3:** Styling and layout design.
- **JavaScript:** Core calculation logic and DOM manipulation.

## 📂 How to Run
1. Clone this repository: `git clone https://github.com/Deepxk17/simple-calculator.git`
2. Open `index.html` in your favorite web browser.
